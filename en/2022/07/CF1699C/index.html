Problem Statement

Given a permutation aaa of length nnn, find how many permutations bbb of the same length are similar to aaa.

If, for every interval [l,r](1≤l≤r≤n)[l, r] (1 \le l \le r \le n)[l,r](1≤l≤r≤n), the following condition holds:

MEX⁡(al,al+1,…,ar)=MEX⁡(bl,bl+1,…,br)\operatorname{MEX}(a_l, a_{l + 1}, \ldots ,a_r) = \operatorname{MEX}(b_l, b_{l + 1}, \ldots ,b_r) MEX(al​,al+1​,…,ar​)=MEX(bl​,bl+1​,…,br​)

then permutations aaa and bbb are called similar.

Here, for an array ccc, MEX⁡\operatorname{MEX}MEX is defined as the smallest non-negative integer xxx that does not appear in ccc.

For example, MEX⁡([1,2,3,4,5])=0\operatorname{MEX}([1,2,3,4,5]) = 0MEX([1,2,3,4,5])=0 and MEX⁡([0,1,2,4,5])=3\operatorname{MEX}([0,1,2,4,5]) = 3MEX([0,1,2,4,5])=3.

Because the answer may be large, print it modulo 109+710^9 + 7109+7.
