重心坐标

和 GAMES101 课程中讲的一样，重心坐标（特别是三角形的）在图形学中非常有用，可以很方便的把三角形节点上的信息插值到三角形面上。

我最初听课的时候还是有较多疑惑的，比如为什么一定要三个系数和相加等于 1，并且三个系数都不为负数，某个点才是在三角形内部的呢？

后来思考了一段时间，感觉用“顾名思义”的方式理解这个概念是直观的，也就是从重心的角度去理解。

物理角度

假设有一个三角形，其三个顶点的质量分别为 Ma,Mb,McM_a, M_b, M_cMa​,Mb​,Mc​，并且除了顶点外，别的区域质量均为 0，那么根据重心的定义，三角形的重心就是：

重心=MaA+MbB+McCMa+Mb+Mc\text{重心} = \frac{M_aA + M_bB + M_cC}{M_a + M_b + M_c} 重心=Ma​+Mb​+Mc​Ma​A+Mb​B+Mc​C​

把这三个项分开，可以得到：

AMaMa+Mb+Mc+BMbMa+Mb+Mc+CMcMa+Mb+McA\frac{M_a}{M_a + M_b + M_c} + B\frac{M_b}{M_a + M_b + M_c} + C\frac{M_c}{M_a + M_b + M_c} AMa​+Mb​+Mc​Ma​​+BMa​+Mb​+Mc​Mb​​+CMa​+Mb​+Mc​Mc​​

这个形式简直和重心坐标的 αA+βB+γC\alpha A + \beta B + \gamma CαA+βB+γC 太相似了，每项都是一个系数乘以一个顶点的坐标。

观察形如 MaMa+Mb+Mc\frac{M_a}{M_a + M_b + M_c}Ma​+Mb​+Mc​Ma​​ 的项，可以发现，计算重心时的系数完全符合重心坐标的要求，即 α+β+γ=1\alpha + \beta + \gamma = 1α+β+γ=1，并且每项不是负数。

现在用物理上重心的思路来思考，我们在转换普通坐标到重心坐标时到底在转换什么？

把重心公式的系数对应到重心坐标的系数，也就是 MaMa+Mb+Mc→α\frac{M_a}{M_a + M_b + M_c} \to \alphaMa​+Mb​+Mc​Ma​​→α，我们能发现，这些系数实际上是三角形每个顶点的质量占总质量的比例。

也就是说，在转换的过程中（假设把 ppp 点从笛卡尔坐标系转换到重心坐标系），我们实际上计算的是这样一个问题：三角形的三个顶点如何分配重量，才能使得三角形的重心位于 ppp 点？

转换后的坐标，(α,β,γ)(\alpha, \beta, \gamma)(α,β,γ) 其实就是在三角形三个顶点上分配的重量的比例。

这样一来，我们就很容易理解这三个数为什么符合相加为 1 并且非负才能使点在三角形内部的要求了。首先，从物理的角度来思考，如果质量非负，那么重心一定在物体的内部。其次，如果要符合重心的定义，那么 α,β,γ\alpha, \beta, \gammaα,β,γ 相加一定是等于 1 的。因为它们三个代表的是三个顶点的质量占总质量的比例，所以这三个数字相加一定是 1（对应总质量）。

当然，重心坐标是由物理上的定义推广出来的，所以用纯代数的方法来说明可能更有说服力。

代数角度

我们知道，三角形的重心坐标就是三个顶点的线性组合来标记一个坐标，也就是 p=αA+βB+γCp = \alpha A + \beta B + \gamma Cp=αA+βB+γC。现在我们可以尝试用代数的方式来说明，为什么这三个系数相加等于 1 并且非负才能使点在三角形内部。

首先，我们可以把 ppp 点用另一种方式表示出来，这对之后的推导会有帮助：

p=A+uAB→+vAC→p = A + u\overrightarrow{AB} + v\overrightarrow{AC} p=A+uAB+vAC

注意这里的 uuu 和 vvv 其实和重心坐标的三个系数没有关系，这个式子的含义是：从 A 点出发，沿着 AB 方向走 uuu 个单位，然后沿着 AC 方向走 vvv 个单位，就可以到达 ppp 点。

首先，如果要让点在三角形内部，有一点是可以确定的：uuu 和 vvv 是非负的，因为考虑当前在 A 点上，只向着 BA→\overrightarrow{BA}BA 或者 CA→\overrightarrow{CA}CA 方向走任何距离，马上就会走出三角形。

对上式稍微做一点变形，可以得到：

p−A=uAB→+vAC→pA→=uAB→+vAC→uAB→+vAC→−pA→=0p - A = u\overrightarrow{AB} + v\overrightarrow{AC} \\ \overrightarrow{pA} = u\overrightarrow{AB} + v\overrightarrow{AC} \\ u\overrightarrow{AB} + v\overrightarrow{AC} - \overrightarrow{pA} = {0} p−A=uAB+vACpA​=uAB+vACuAB+vAC−pA​=0

可以这么理解这个式子：我们把包括 ppp 点在内的所有点都通过 −OA→-\overrightarrow{OA}−OA 移动了一个距离，这个时候 AAA 一定在原点。

这样这个式子就很直观了：先通过 uAB→+vAC→u\overrightarrow{AB} + v\overrightarrow{AC}uAB+vAC 这个向量从原点走到 ppp 点，再从 ppp 点用 pA→\overrightarrow{pA}pA​ 走回 AAA，也就是原点。

因为这两个部分方向相反并且大小相同（一个从 AAA 走到 ppp，一个从 ppp 走到 AAA），所以这两个向量相加一定是 0。

拆开这个式子，再重新整理，可以得到：

u(B−A)+v(C−A)+A−p=0uB−uA+vC−vA+A−p=0A(1−u−v)+uB+vC−p=0(1−u−v)A+uB+vC=p\begin{align*} u(B - A) + v(C - A) + A - p &= 0 \\ uB - uA + vC - vA + A - p &= 0\\ A(1 - u - v) + uB + vC - p &= 0 \\ (1 - u - v)A + uB + vC &= p \end{align*} u(B−A)+v(C−A)+A−puB−uA+vC−vA+A−pA(1−u−v)+uB+vC−p(1−u−v)A+uB+vC​=0=0=0=p​

这个形式完全和重心坐标一致，所以我们可以确定：α=1−u−v\alpha = 1 - u - vα=1−u−v，β=u\beta = uβ=u，γ=v\gamma = vγ=v。

把这些东西相加，可以发现 1−u−v+u+v=11 - u - v + u + v = 11−u−v+u+v=1，确实，重心坐标的三个系数相加一定是 1。

前面我们已经说明了，uuu 和 vvv 是非负的，但如果 u+v>1u + v > 1u+v>1，α\alphaα 不就是负数了吗？

考虑下图，AMAMAM 是 AAA 到 BCBCBC 的一条垂线（MMM 不是 BCBCBC 的中点，这里不用等边三角形可能更清晰，但是我懒）：

回到最初的定义中，也就是 p=A+uAB→+vAC→p = A + u\overrightarrow{AB} + v\overrightarrow{AC}p=A+uAB+vAC，uvuvuv 分别是朝 AB→\overrightarrow{AB}AB 和 AC→\overrightarrow{AC}AC 方向走过的距离。

那么 uAB→u\overrightarrow{AB}uAB 投影在 AM→\overrightarrow{AM}AM 上的长度就是 uAB→⋅AM→∣AM→∣u\overrightarrow{AB} \cdot \overrightarrow{AM} \over |\overrightarrow{AM}|∣AM∣uAB⋅AM​

观察上图可以发现，如果 uuu 为 1，也就是 AB→\overrightarrow{AB}AB 有完整的长度，这个投影的长度一定等于 ∣AM→∣|\overrightarrow{AM}|∣AM∣。同样的，如果 uuu 为 0，投影在 AM→\overrightarrow{AM}AM 方向的长度也一定是 0。

因为投影，或者说点乘是一个线性的操作，所以如果 uuu 为 0.5，投影的长度一定是 ∣AM→∣|\overrightarrow{AM}|∣AM∣ 的一半。

因此我们可以说，AB→\overrightarrow{AB}AB 在 AM→\overrightarrow{AM}AM 上的投影长度是 u∣AM→∣u|\overrightarrow{AM}|u∣AM∣

相同的，AC→\overrightarrow{AC}AC 在 AM→\overrightarrow{AM}AM 方向的投影长度符合一样的规则。

为了让 ppp 点留在三角形内部，uAB→+vAC→u\overrightarrow{AB} + v\overrightarrow{AC}uAB+vAC 这个向量投影在 AM→\overrightarrow{AM}AM 上的长度必须小于等于 AM→\overrightarrow{AM}AM 本身的长度，不然 ppp 点就会从 BCBCBC 这条边跑出三角形。

AB→\overrightarrow{AB}AB 在 AM→\overrightarrow{AM}AM 上的投影长度是 u∣AM→∣u|\overrightarrow{AM}|u∣AM∣，AC→\overrightarrow{AC}AC 在 AM→\overrightarrow{AM}AM 上的投影长度是 v∣AM→∣v|\overrightarrow{AM}|v∣AM∣，自然，这两个向量的和在 AM→\overrightarrow{AM}AM 上的投影长度就是 (u+v)∣AM→∣(u + v)|\overrightarrow{AM}|(u+v)∣AM∣。

前面说过，为了使得 ppp 点留在三角形内部，uAB→+vAC→u\overrightarrow{AB} + v\overrightarrow{AC}uAB+vAC 这个向量投影在 AM→\overrightarrow{AM}AM 上的长度必须小于等于 AM→\overrightarrow{AM}AM 本身的长度，也就是 (u+v)∣AM→∣≤∣AM→∣(u + v)|\overrightarrow{AM}| \le |\overrightarrow{AM}|(u+v)∣AM∣≤∣AM∣，那么 u+v≤1u + v \le 1u+v≤1。

因为 α=1−u−v\alpha = 1 - u - vα=1−u−v，并且 u+v≤1u + v \le 1u+v≤1，所以我们可以说明，α\alphaα 也一定是非负的。

至此我们已经能说明，为什么一个在三角形内部的点就要符合 α+β+γ=1\alpha + \beta +\gamma = 1α+β+γ=1，并且每个系数非负了。当然要把一个平面直角的坐标转换到重心坐标，还是需要一些相对复杂的计算的。这部分的内容，我觉得这篇博客中介绍的第一种方法相对易于理解并且很巧妙，有兴趣的可以看下。
